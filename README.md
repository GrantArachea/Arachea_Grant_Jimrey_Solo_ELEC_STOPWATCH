# Solo ELEC Stopwatch

A simple **web-based stopwatch application** built with **HTML**, **CSS**, and **JavaScript** that lets users measure elapsed time interactively.

This project was created as a **solo assignment** for the **ELEC3.

## Features

- Start, stop, and reset elapsed time  
- Displays time in **minutes**, **seconds**, and optionally **hours**  
- Event-driven JavaScript logic with DOM updates for responsive UI  
- Simple and clean front-end design

The functionality is similar to simple web stopwatch projects built with HTML/CSS/JS, allowing users to measure incrementing time and reset as needed.:contentReference[oaicite:0]{index=0}

## Demo

To view and use the stopwatch, open `index.html` in your browser after setup.

## Technologies Used

- **HTML** – Page structure  
- **CSS** – Visual styling and layout  
- **JavaScript** – Stopwatch timing logic and user interaction

## Getting Started

Follow the instructions below to run this project locally.

### Prerequisites

You only need a modern web browser (Chrome, Firefox, Edge, Safari) to run and test this project.

### Installation

1. **Clone the repository**

    ```bash
    git clone https://github.com/GrantArachea/Arachea_Grant_Jimrey_Solo_ELEC_STOPWATCH.git
    ```

2. **Navigate to project folder**

    ```bash
    cd Arachea_Grant_Jimrey_Solo_ELEC_STOPWATCH
    ```

3. **Open the application**

    Open the `index.html` file in your browser by double-clicking it or by using *File → Open* in your browser.

## Usage

- Click **Start** to begin the stopwatch.  
- Click **Stop** to pause the elapsed time.  
- Click **Reset** to clear the time back to zero.

The elapsed time will update in real time as the stopwatch runs.
